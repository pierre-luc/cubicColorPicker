CubicColorPicker
================
Demo
----

[click here to see the demo](http://codepen.io/pierre-luc/pen/GqyxqG)

Usage
-----
```javascript
var picker = new CubicColorPicker(element);

$(picker).on('colorHasChanged', function(){
  picker.copyColor(); // copy the color into the clipboard
});
```
